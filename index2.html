Concept Notes

      

        
Python

        
  • Everything in Python is an object.
  • Functions are defined using def.
  • Lists are dynamic arrays.

        
JavaScript

        
  • JS is single-threaded but asynchronous.
  • DOM manipulation is core for UI.
  • Use const/let instead of var.

        
Java

        
  • Strictly typed OOP language.
  • Supports multithreading.
  • Everything runs inside classes.

        
C++

        
  • Combines OOP and procedural features.
  • Use templates for generic programming.
  • Supports operator overloading.

        
C#

        
  • Developed by Microsoft for .NET.
  • Supports LINQ and async/await.
  • Uses garbage collection.

        
Go

        
  • Fast, compiled, and concurrent.
  • Goroutines enable parallel execution.
  • Memory-safe and efficient.
